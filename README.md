# Mullick Law - Website Redesign

This project is a redesign of the Mullick Law website, built with **React**, **Vite**, **Tailwind CSS**, and **Shadcn UI**.

## Project Overview

The goal of this project is to create a modern, professional, and accessible website for the Law Office of Sundeep K. Mullick, P.A.

## Getting Started

To run the project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/codewithriad/mullick-law.git
    cd mullick-law
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

## Technologies Used

- [Vite](https://vitejs.dev/) - Frontend Tooling
- [React](https://react.dev/) - UI Library
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Shadcn UI](https://ui.shadcn.com/) - Reusable components built with Radix UI and Tailwind CSS
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript

## Deployment

This project can be easily deployed to Vercel, Netlify, or any static site hosting service.

1.  Push your code to a Git repository.
2.  Connect your repository to your hosting provider.
3.  The build settings should be automatically detected (Build command: `npm run build`, Output directory: `dist`).
